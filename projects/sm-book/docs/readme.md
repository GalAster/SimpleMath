## DataType

| Type     | Example             | Remark           |
| :------- | :------------------ | :--------------- |
| Integer  | `42`                | Atom, BigInteger |
| Decimal  | `1.234`             | Atom, BigDecimal |
| String   | `'literal'`         | Atom             |
| Cite     | `$cite.path`        | Atom             |
| Special  | `true`, `null`      | Atom             |
| List     | `[1, 2, 3]`         | Container        |
| Dict     | `{1: 3}`            | Container        |
| DictMark | `[dict.path]`       | Scope            |
| ListMark | `<list.path>`       | Scope            |
