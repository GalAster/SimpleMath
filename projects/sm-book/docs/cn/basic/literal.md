


## Numbers

- 浮点输入: `**`
  - `2**5` => `2e5`
- 精度/精密度: `¯`
  - `1¯20` Precision
- 准度/准确度: `¨`
  - `1¨20` Accuracy

## REPL only


¶ 1 + 1
⁋ 2

- 输入
  - 上个输入 `¶`
  - 第 n 个输入 `¶n`
  - 上上个输入 `¶¶`
  - 上上上个输入 `¶¶¶` 以此类推
- 输出
  - 上个输出 `⁋`
  - 第 n 个输出 `⁋n`
  - 上上个输出 `⁋⁋`
  - 上上上个输出 `⁋⁋⁋` 以此类推
